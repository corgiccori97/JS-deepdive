## :bulb: 함수 리터럴

- 함수는 객체지만 일반 객체와는 다른 고유한 프로퍼티를 가짐
-

## :bulb: 함수 선언

:one: 함수 선언문

```
function add(x, y) {
    return x + y;
}
```

:two: 함수 표현식

```
var add = function (x, y) {
    return x + y;
};
```

:three: Function 생성자 함수

```
var add = new Function('x', 'y', 'return x + y');
```

:four: 화살표 함수(ES6)

```
var add = (x, y) => x + y;
```
